The Retiga E7 is a scintillator-coupled CMOS camera used for tomography, radiography, and near-field HEDM.

The Retiga E7 computer can be accessed from the station computer by double-clicking the "e7-p30.nxs" shortcut on the desktop.

More instructions for starting the IOC, etc can be found [here](https://cornell.box.com/s/no6s0r8lz1gys9uheyb086owffuve7ya).
